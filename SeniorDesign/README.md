The senior design project to which I was apart of the group is too large for me to host here so below I have provided a link to the gitlab repo

https://git.ece.iastate.edu/isu-chip-fab/sdmay26-23

My specific contributions to the project was helping in design for the opamp, providing support with tooling including making scripts to streamline specific repetetive processes as well as doing data processing and visualisation for the team, and finally I worked on documentation including how to use the open source tools for more advanced simulations. The two main gains of the project were an introduction to the open source tools that allow students to work with a real fabricatable PDK, and to experience everything that comes with it. There were numerous trials and tribulations just getting the tools to work for each of the students. The design process was tricky as we were simultaneously learning a lot of the thing we needed in our EE435 class while we were working on implementation in the PDK. The project was fairly ambitous for out group, and I think we got to a reasonably complete place. The primary goals of the project were an educational pursuit wherein future students could look back on our designs and the documentation we created in order to come up to speed with the tools faster than we were, and hopefully complete more ambitious analog projects within the PDK. Furthermore the project sparked enough interest for me that I have been, as a hobby, working on other basic analog and mixed signal designs with the tools.
