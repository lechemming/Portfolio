# Sub Nanosecond pulse generator for TDR

This was a project for my high speed systems class, the project was very open ended and the choice to make
a pulse generator was a specificc choice of my own. It involved numerous hours of research, schematic building,
selecting components for my BOM, layout, and assembly. I took on the project alone and was able to explore
pcb design for a relatively high speed design and learn some of the nuances involved in that.

The project was designed using KiCAD and was fabricated by JLC PCB. This was only my second time doing a project like this to completion, and the first time doing it without a specific reference design that I was essentially copying. It was a good practical introduction into producing my own PCB's and making calls about how I should implement my design in a cost effective way while fitting my specification. It also required me to make my own and make modification to device footprints and read a lot about implementation of specific IC's within datasheets.
